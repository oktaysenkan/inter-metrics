# inter-metrics

[Dynamic Metrics](https://rsms.me/inter/dynmetrics/) calculator for Inter font

## Installation

```sh
npm install inter-metrics
```

## Usage

```js
import { textStyle } from "inter-metrics";


textStyle(16);

// {
//   "fontSize": 16,
//   "letterSpacing": -0.010959778564167367,
//   "lineHeight": 22.4,
// }
```
